Mi Portafolio

El portafolio fue desarrollado utilizando HTML, CSS y JavaScript con el objetivo de presentar tus proyectos y habilidades en una página web atractiva y fácil de navegar. En primer lugar, se utilizó HTML para estructurar la página web y dividirla en secciones que contienen información sobre ti, tus habilidades y tus proyectos. Esto incluyó la creación de secciones como "Acerca de mí", "Habilidades" y "Proyectos". A continuación, se utilizó CSS para diseñar y personalizar el estilo visual del sitio web. Esto incluyó la elección de colores, tipografía, tamaño y diseño de los elementos de la página web. También se utilizó CSS para crear efectos de animación y transiciones para mejorar la experiencia del usuario. Finalmente, se utilizó JavaScript para agregar interactividad y funcionalidad a la página web.
